# Level 0 Cell 制作指南

本文用于制作 BackroomsCore Level 0 的 FAWE / WorldEdit schematic 房间模板。

当前 Level 0 不做真正无限迷宫，而是采用有限区域拼接：

```text
一个生成区域 = N x N 个 cell
一个 cell = 16 x 16 x 6
一个房间模板 = 一个或多个 cell 的 schematic
生成器 = 先生成连通图，再根据连接口、权重、标签选择模板并粘贴
```

## 1. 标准 cell 尺寸

第一阶段所有模板优先做 `1x1`：

```text
尺寸：16 x 16 x 6
地板：y=0
内部可用高度：y=1 到 y=4
天花板：y=5
墙厚：1
门洞：3 宽 x 3 高
```

门洞固定在四边中点：

```text
north 门：z=0 墙上，x=6,7,8，y=1,2,3
south 门：z=15 墙上，x=6,7,8，y=1,2,3
west 门：x=0 墙上，z=6,7,8，y=1,2,3
east 门：x=15 墙上，z=6,7,8，y=1,2,3
```

所有模板必须严格遵守门洞位置，否则旋转和拼接后会错位。

## 2. 当前 marker / 占位方块

当前 CraftEngine 方块是否能稳定保存进 schematic 还需要实机验证，所以第一阶段使用原版方块作为 marker。

这些 marker 是给插件扫描用的，不一定是最终玩家看到的方块。

| 原版 marker | 配置 key | 当前意义 | 后续目标 |
|---|---|---|---|
| `YELLOW_CARPET` | `resource` | Level 0 地毯资源点 | 替换为 `backrooms:loose_carpet_cache` |
| `SEA_LANTERN` | `light` | 荧光灯 marker | 替换为 `backrooms:flickering_fluorescent_light` |
| `BARREL` | `loot` | 物资箱 / loot 点 | 替换或注册为 `backrooms:supply_crate` |
| `LODESTONE` | `transition` | 真出口 / 切层触发点 | 注册 transition，可能替换为空气或 CE 标记 |
| `IRON_TRAPDOOR` | `maintenance-door` | 维护门 marker | 替换为 `backrooms:maintenance_door_marker` |
| `CHISELED_STONE_BRICKS` | `stairwell` | 楼梯井视觉 marker | 替换为 `backrooms:stairwell_marker` |

重要区分：

```text
假楼梯井：可以有 CHISELED_STONE_BRICKS，但不要放 LODESTONE
真楼梯井：必须有 CHISELED_STONE_BRICKS + LODESTONE
```

## 3. Level 0 推荐 cell 类型总表

| 类型 | 建议数量 | 主要功能 | 出现频率 |
|---|---:|---|---|
| `common_room` | 8-12 | Level 0 主体黄墙纸房间 | 最高 |
| `straight_corridor` | 4-6 | 拉长空间，制造走廊感 | 高 |
| `corner_corridor` | 4-6 | 打断视线，制造迷路 | 中高 |
| `t_junction` | 3-4 | 路线选择，形成分支 | 中 |
| `cross_junction` | 2-3 | 少量开放交叉路口 | 低 |
| `dead_end` | 4-6 | 资源、假线索、恐怖点 | 中 |
| `office_cluster` | 3-5 | 办公区变化 | 中低 |
| `maintenance_room` | 2-4 | 维护间、工具资源、路线暗示 | 低 |
| `false_stairwell` | 2-4 | 假出口，欺骗玩家 | 很低 |
| `stairwell_exit` | 1-2 | 真正通往 Level 1 的出口 | 极低 |
| `anomaly_room` | 2-4 | 稀有异常空间 | 极低 |
| `supply_room` | 2-3 | 小补给间 | 低 |
| `dark_room` | 2-3 | 暗房、恐怖氛围 | 低 |

第一阶段最低推荐制作：

```text
common_room_01
common_room_02
straight_corridor_01
corner_corridor_01
t_junction_01
dead_end_cache_01
false_stairwell_01
stairwell_exit_01
```

更理想的第一批模板：

```text
common_room_01
common_room_02
common_room_03
common_room_04

straight_corridor_01
straight_corridor_02

corner_corridor_01
corner_corridor_02

t_junction_01
cross_junction_01

dead_end_cache_01
dead_end_dark_01

maintenance_room_01
false_stairwell_01
stairwell_exit_01
```

## 4. common_room 普通黄墙纸房间

### 功能

Level 0 的主体。大部分区域应该由普通黄墙纸房间和走廊组成。

### 建议数量

```text
8-12 个变体
```

### 推荐 connectors

可以做：

```text
四通：north, east, south, west
三通：north, east, south
两通：north, south
转角：north, east
```

### marker 建议

| marker | 推荐数量 | 用途 |
|---|---:|---|
| `SEA_LANTERN` | 1-3 | 天花板荧光灯 |
| `YELLOW_CARPET` | 0-1 | 松动地毯资源点 |
| `BARREL` | 0-1 | 偶尔的小补给 |
| `IRON_TRAPDOOR` | 0-1 | 假维护门 |
| `LODESTONE` | 0 | 普通房不要放真出口 |
| `CHISELED_STONE_BRICKS` | 0 | 普通房不要放楼梯井 |

### 变体建议

```text
common_room_01：普通四通黄墙纸房间
common_room_02：墙纸轻微破损
common_room_03：灯更少，局部偏暗
common_room_04：柱子错位，阻挡视线
common_room_05：地毯颜色偏差
common_room_06：墙上有假门痕迹
common_room_07：重复墙纸图案异常
common_room_08：低矮隔断
```

## 5. straight_corridor 直走廊

### 功能

拉长空间，让玩家产生“走了很久”的感觉。

### 推荐 connectors

```text
north, south
```

配置中允许旋转后成为：

```text
east, west
```

### marker 建议

| marker | 推荐数量 | 用途 |
|---|---:|---|
| `SEA_LANTERN` | 2-4 | 走廊灯 |
| `YELLOW_CARPET` | 0-1 | 少量资源点 |
| `IRON_TRAPDOOR` | 0-1 | 假维护门 |
| `BARREL` | 0 | 走廊不建议频繁放 loot |
| `LODESTONE` | 0 | 不放出口 |
| `CHISELED_STONE_BRICKS` | 0 | 不放楼梯井 |

### 变体建议

```text
straight_corridor_01：普通直走廊
straight_corridor_02：暗直走廊
straight_corridor_03：灯管缺失
straight_corridor_04：一侧有假门
```

## 6. corner_corridor 转角

### 功能

打断视线，让玩家无法一眼看穿地图。

### 推荐 connectors

```text
north, east
```

配置中允许旋转成其他方向的转角。

### marker 建议

| marker | 推荐数量 | 用途 |
|---|---:|---|
| `SEA_LANTERN` | 1-2 | 转角灯 |
| `IRON_TRAPDOOR` | 0-1 | 维护门 |
| `YELLOW_CARPET` | 0-1 | 角落资源 |
| `BARREL` | 0-1 | 少量补给 |
| `LODESTONE` | 0 | 不放出口 |
| `CHISELED_STONE_BRICKS` | 0 | 不放楼梯井 |

## 7. t_junction T 字路口

### 功能

形成路线选择，让玩家开始迷路。

### 推荐 connectors

例如：

```text
north, east, west
```

或者：

```text
north, east, south
```

### marker 建议

| marker | 推荐数量 | 用途 |
|---|---:|---|
| `SEA_LANTERN` | 1-2 | 中心灯 |
| `IRON_TRAPDOOR` | 0-1 | 一侧维护门 |
| `YELLOW_CARPET` | 0-1 | 分支角落资源 |
| `BARREL` | 0-1 | 低概率补给 |
| `LODESTONE` | 0 | 不放出口 |
| `CHISELED_STONE_BRICKS` | 0 | 不放楼梯井 |

### 功能

### 推荐 connectors

### marker 建议

| marker | 推荐数量 | 用途 |
|---|---:|---|
| `SEA_LANTERN` | 1-2 | 中心灯或坏灯 |
| `YELLOW_CARPET` | 0-1 | 小资源点 |
| `IRON_TRAPDOOR` | 0-1 | 维护门 |
| `BARREL` | 0 | 不建议频繁放 loot |
| `LODESTONE` | 0 | 不放出口 |
| `CHISELED_STONE_BRICKS` | 0 | 不放楼梯井 |

## 9. dead_end 死路

### 功能

死路用于奖励、恐怖、假线索。

### 推荐 connectors

只有一个方向，例如：

```text
south
```

### marker 建议

| marker | 推荐数量 | 用途 |
|---|---:|---|
| `YELLOW_CARPET` | 1-2 | 松动地毯资源点 |
| `BARREL` | 0-1 | 补给箱 |
| `IRON_TRAPDOOR` | 0-1 | 假维护门 |
| `SEA_LANTERN` | 0-1 | 暗灯 |
| `CHISELED_STONE_BRICKS` | 0-1 | 仅假楼梯井死路使用 |
| `LODESTONE` | 0 | 普通死路不要放真出口 |

### 变体建议

```text
dead_end_cache_01：资源死路，放 YELLOW_CARPET
dead_end_crate_01：补给死路，放 BARREL
dead_end_dark_01：暗死路，无奖励或很少奖励
dead_end_false_door_01：假维护门死路，放 IRON_TRAPDOOR
```

## 10. maintenance_room 维护间

### 功能

功能房，放维修工具、fuse、toolbox、路线暗示。

### 推荐 connectors

```text
一个入口或两个入口
```

### marker 建议

| marker | 推荐数量 | 用途 |
|---|---:|---|
| `IRON_TRAPDOOR` | 1-2 | 维护门 marker |
| `BARREL` | 1-2 | 工具箱 / 物资箱 |
| `SEA_LANTERN` | 1 | 工作灯 |
| `YELLOW_CARPET` | 0-1 | 少量资源 |
| `LODESTONE` | 0-1 | 未来机器事件点，谨慎使用 |
| `CHISELED_STONE_BRICKS` | 0 | 一般不放 |

## 11. false_stairwell 假楼梯井

### 功能

欺骗玩家，让玩家以为找到出口。

### 推荐 connectors

通常一个入口：

```text
south
```

### marker 建议

| marker | 推荐数量 | 用途 |
|---|---:|---|
| `CHISELED_STONE_BRICKS` | 1-3 | 楼梯井视觉 marker |
| `IRON_TRAPDOOR` | 1 | 假门 |
| `SEA_LANTERN` | 0-1 | 暗灯 |
| `BARREL` | 0-1 | 安慰奖励 |
| `YELLOW_CARPET` | 0-1 | 少量资源 |
| `LODESTONE` | 0 | 假出口绝对不要放 |

关键规则：

```text
false_stairwell 可以放 CHISELED_STONE_BRICKS
false_stairwell 不要放 LODESTONE
```

## 12. stairwell_exit 真出口

### 功能

真正通往 Level 1 的出口。

### 推荐 connectors

通常一个入口，例如：

```text
south
```

### 必放 marker

| marker | 推荐数量 | 用途 |
|---|---:|---|
| `CHISELED_STONE_BRICKS` | 1-3 | 楼梯井标记 |
| `LODESTONE` | 1 | 真 transition marker |
| `IRON_TRAPDOOR` | 1 | 出口门 / 维护门 |
| `SEA_LANTERN` | 1 | 特殊灯 |
| `BARREL` | 0-1 | 可选出口补给 |
| `YELLOW_CARPET` | 0 | 不重要 |

关键规则：

```text
真出口 = CHISELED_STONE_BRICKS + LODESTONE
假出口 = CHISELED_STONE_BRICKS，但没有 LODESTONE
```

| `YELLOW_CARPET` | 0-1 | 异常资源 |

## 15. 建图原则

Level 0 的重点不是复杂结构，而是：

```text
重复
轻微差异
低频功能房
稀有异常
真出口极少
假出口存在
```

推荐比例：

```text
70% 普通房间 / 走廊
15% 轻微变化
10% 功能房 / 资源房
4% 异常房
1% 真出口
```

不要每个房间都放资源或箱子。  
Level 0 应该是探索、迷路、紧张，而不是刷箱子地图。
